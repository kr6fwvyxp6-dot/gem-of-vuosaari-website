Husregler, villkor och avbokning

Allt som är bra att veta före din vistelse.

Husregler

  • Incheckning: från kl. 15:00 · Utcheckning: senast kl. 11:00
  • Max antal gäster: 4 gäster
  • Rökning: ej tillåten inomhus; tillåten endast på verandan vid lägenhetens entré
  • Husdjur: ej tillåtet
  • Fester eller evenemang: ej tillåtet
  • Tysta timmar: 22:00–08:00, av hänsyn till grannarna

Villkor

Bokning och betalning

Full betalning ska erläggas innan vistelsen börjar. Vi accepterar banköverföring, kreditkort och faktura. Om du bokar via Airbnb eller Booking.com hanteras betalningen av den plattformen enligt dess egna villkor.

Skador och ansvar

Du ansvarar för eventuella skador på lägenheten, möbler eller utrustning utöver normalt slitage under din vistelse. Ingen separat depositionsavgift tas ut vid bokningen. Om skador uppstår kontaktar vi dig efter utcheckningen med bilder och kostnaden för reparation eller ersättning, och kan debitera detta via betalningsmetoden som användes för din bokning — eller, för bokningar via Airbnb och Booking.com, genom plattformens egen skadeprocess. Vänligen meddela oss om skador så snart de inträffar.

Avbokningspolicy

Fri avbokning upp till 7 dagar före incheckning. Avbokningar som görs mindre än 7 dagar före incheckning, samt uteblivet besök, återbetalas inte. Om du bokade via Airbnb eller Booking.com gäller den plattformens avbokningspolicy istället, eftersom varje plattform tillämpar sina egna regler för bokningar gjorda där.
